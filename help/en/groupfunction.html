Groupfunction

The block

The groupfunction block has various functions. You can count colums. Also you get the average, sum, variance, maximum, minimum and the standard deviation of a column. It is used in the 'SELECT'-section. This block chnages the colour to simbolise the types.

Functionality

Using the Block

To use this block you just plug in the green table and colum block and choose your table and paramter. This is shown in the picture on the left.

Setting an alias

In some functions you have to set an alias. To set the alias, just click in the text field, marked with 'AS' and write it down. The alias is the name under which the result of the function will be shown. You can use it to order or group your results.To use the alias you need the alias variable

Functions

Average

The average function, calculates the average value of a column. Usually it is used with columns, who hold quantities or money(e.g. a Price).

Summarize

The summarize function, calculates the summed value of a column. Usually it is used with columns, who hold quantities or money(e.g. a Price).

Minimum

The minimum function, returns the minimum value of a column. Usually it is used with columns, who hold quantities or money(e.g. a Price).

Maximum

The maximum function, returns the maximum value of a column. Usually it is used with columns, who hold quantities or money(e.g. a Price).

Count

The count function counts the rows of a column and return this as number.

Variance

The variance function returns the population standard variance of a column. It considers rows as the whole population, not as a sample, so it has the number of rows as the denominator.

Standard derivation

The standart derivation function returns the population standard deviation of a column.
