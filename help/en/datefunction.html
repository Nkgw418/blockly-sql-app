Date function

The block

The date function block has varous functions. You can extract, summarize, substract a date. You also can get the systemdate, today and the current date. Extract the date part , the month, the year or the last day of a month of a date value, with the help of this block. You even can convert a date to text.
The block is used with the compare operator and the date value or column.

Functions

The extract function uses a date value or column and a specifier. To use the function you plug the date value or column or systemdate, today, current date into the function.

Systemdate

The sysdate function returns the systemdate. The time when the function is exectuted is called systemdate. It could be used to compare a date column or value with the systemdate.
You also can use it in date functions. The format is : 'YYYY-MM-DD HH:MM:SS'

Now

Like the sysdate function, the now function returns the time, when the function began to execute. It could be used to compare a date column or value with the now.
You also can use it in date functions. The format is : 'YYYY-MM-DD HH:MM:SS'

Current date

Like the sysdate and now function, the current date function returns the time, dependig on the context in which it is used. Its the most common function of this three time functions. It could be used to compare a date column or value with the current date.
You also can use it in date functions. The format depending on the context is : 'YYYY-MM-DD' or YYYYMMDD

Date

The date function extracts the date part of a date time expression, eg. of a date column. It could be used in the compare operation.
You also can use it in date functions.

Last day of month

The last day function returns the last day of a month of the plugged in date value or column. It could be used to compare operator.
You also can use it in date functions.

Month

The last day function returns the month of the plugged in date value or column. It could be used with the compare operator.
You also can use it in date functions.

Year

The last day function returns the month of the plugged in date value or column. It could be used with the compare operator.
You also can use it in date functions.

To string

The to string function is the inverse function of 'TO DATE'. IT uses a date value and a specifier.
