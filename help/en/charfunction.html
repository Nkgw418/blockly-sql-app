Char function

The block

The char function block has various functions. You can get the length,the ascii value and the substring . You also canleft or right trim or left or right pad a text, make a text with lower cases to upper cases or do it the other way around or replace it and search the position of a substring in the text. You can convert a text to date.
It is used with the compare operator and the text value or column.

Functions

ASCII

The ascii function uses a text value or column. It returns a numeric value between 0 and 255 of the text.

Position of substring

The in string function use two text or column values
It returns the position from the second value in the first value.
A simple example is shown below.

Length

The length function uses a text value or column. It returns the length of the text as numeric value

Left and right pad

The left and right pad functions use two text or column values and one numeric value.
They left or right pad the first text value with the second value to the length of the numeric value. The numeric values must plugged into the second input.
A simple example is shown below.

Lower and upper cases

The lower function returns the text value or column with all characters changed to lowercase.
The upper case function does this the other way around.

Left and right trim

The left and right trim function uses a text value or column. It returns the text trimmed left or right.

Replace

The replace function uses three text values or columns. It returns the first value with all occurrences of the second value replaced by the third value.
A simple example is shown below.

Substring

The substring function uses one text value or column and one numeric value, which need to be plugged into the second input.
It returns a substring from first value starting at second value.
It is also possible to use a negative value for the second input. In this case, the beginning of the substring is the absolut number from the end of the string, rather than the beginning.

To date

The to date function is the inverse function of 'TO STRING'. IT uses a text value and a specifier. To use the function you plug in the text type in the function Block. This is shown in the picture below.

To change the default specifier, just click in text field. There are many specifieres.
The “%” character is required before format specifier characters. If you want to learn more about the specifiers, please take a look at the table below.

Specifier

Specifier Description
%a Abbreviated weekday name (Sun..Sat)
%b Abbreviated month name (Jan..Dec)
%c or %m Month, numeric (0..12)
%D Day of the month with English suffix (0th, 1st, 2nd, 3rd, …)
%d or %e Day of the month, numeric (00..31)
%f Microseconds (000000..999999)
%H or %k Hour (00..23)
%h or %I or %l Hour (01..12)
%i Minutes, numeric (00..59)
%j Day of year (001..366)
%M Month name (January..December)
%p AM or PM
%r Time, 12-hour (hh:mm:ss followed by AM or PM)
%S or %s Seconds (00..59)
%T Time, 24-hour (hh:mm:ss)
%U Week (00..53), where Sunday is the first day of the week
%u Week (00..53), where Monday is the first day of the week
%V Week (01..53), where Sunday is the first day of the week;used %X
%v Week (01..53), where Sunday is the first day of the week;used %x
