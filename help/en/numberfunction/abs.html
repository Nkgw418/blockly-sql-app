Number function

The block

The number function block has varius functions. You can get the absolute number or the square root of a numeric value. You can power, round, sign, modulo a numeric value. It ist used with the compare operator. The value you use with the function are type numeric.

Functions

Absolut

The absolute function, returns the absolut value of a numeric variable or column.

Square root

The square root function returns the square root of a nonnegative number variable .

Round

The round function rounds a numeric value or column with a specified number po decimal places.

In the function you can specify this decimal places by a numeric value.

Power

The power function returns a numeric value or column raised to the power of a numeric value.

Sign

The sign function returns the sign of a numeric value. For example if your value is '-17' it will return -1 or if it is '234' it will return 1. Notice: If your value is '0' it will return '0'.

Modulo

The modulo function returns the remainder of a numeric value or column divided by a numeric value.
