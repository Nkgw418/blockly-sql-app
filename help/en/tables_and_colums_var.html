Tables and columns(variable)

The block

This block symbolises the tables and columns of your database. It is used as a value in the "WHERE"-section and will be used with the compare and the mathematical operator. Also it will be used with functions for number types, date types and text types, corresponding to the type of this block.
There are four colours, symbolizing the type of the column, in this block. If you want to learn more about types, please see 'The types'.

Functionality

This block has two dropdowns. With the left dropdown you can choose the table and with the right you can choose the column.

The types

This colour,violet, symbolises the numeric values of the database

This colour, brown, symbolises the textual values of the database

This colour,tapestry, symbolises the date values of the database

This colour ,cyan, symbolises the boolean values of the database
