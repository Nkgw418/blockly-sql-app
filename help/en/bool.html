Boolean variable

The block

This block symbolises boolean variables. It is used as a value compare operator, in the "IF" and "IF NULL" of the otherfunction and can be used directly with 'AND'-block or 'OR'-block .

The value of an boolean variable is true or false. Symbolized as in the picture to the right.

Functionality

The default value of this variable is 1.

As you see in the picture on the right, this block has an included dropdown field, where you can change the value.
