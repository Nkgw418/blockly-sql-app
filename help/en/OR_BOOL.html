OR

The block

The 'OR'-block symbolises the 'OR'-statement in SQL. The 'OR' operator displays a record if either the first condition 'OR' the second condition is true. It is used with 'AND'
It can be extended by clicking on the star-icon.

Functionality

When you click on the star-icon, a new workspace appears.

In this workspace you can drop blocks into an 'ADD'-block to expand the 'OR'-block. So for each droped 'OR'-block one new 'OR' appears.Like you see in the picture above.
If you want to reduce the number of or's, you click on the star-icon and pull the 'OR'-blocks out of the 'ADD'block. As an input you use the compare operator.
