Tables and columns(list)

The block

This block symbolises the tables and columns of your database. The '*' in the columnpart of the Block has the same meaning as 'all'. It symbolizes a complete list of the columns in this table. The block is used as a list in the "SELECT"-, "GROUP BY"- and "ORDER BY"-section: It is also used with the groupfunction, the otherfunction and "DISTINCT".

Functionality

The default values in this block are, your first table in the database in the table-part and the '*' in the column-part. In this block are two dropdowns. In the left dropdown you can choose the table as you see in the picture below. In the right you can choose the column.

The types

This colour ,apple green, symbolises a list of columns of the database

This colour,violet, symbolises the numeric values of the database

This colour, brown, symbolises the textual values of the database

This colour,tapestry, symbolises the date values of the database

This colour ,cyan, symbolises the boolean values of the database
