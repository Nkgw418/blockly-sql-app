TO

The block

This block symbolises the 'TO' of a 'SET TO' expression.
It is used as a to in the 'UPDATE'-block and the 'INSERT'-block. You use this block in the 'SET'-section of the corresponding Block. It changes colour to express specified types.

Functionality

The blocks, which you can plug into the first input of the'TO'-block, are table and column variables .
Be sure to use only the tables and column variables, that you have used in the 'INSERT' or 'UPDATE'-section.
In the second input you can plug, every value type, numberfunctions, datefunctions, charfunctions.
