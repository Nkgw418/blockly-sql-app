The block

This blcok block has varius functions. You can test if somthing is true or false,if somthing is null or if somthing is You can greater or, less. It ist used in the 'Select'-section. The values you use with the function are the subselect, the green table and column block, the table and column value block, the compare operator, all type variables. Exept of the if function, all the functions changes the colour to simbolise the type. Notice: The if is always cyan, symbolising type bool.

Functions

IF

The if function uses as first value every time the compare operator.
Its the Statement you want to be true or false. The second input is the value, which will be choosed if the statement is true, the second if the statement is false.
You use this function, if you want to search somthing in the "SELECT"-Statement.

Greater and less

The greater and the least function use as value the green table and column block. The mimum number of values you have to plug in is two.

If null

The if null function uses as first value either one of the groupfunctions, a subselect or the green table column block.
Its the Statement you want to be null. The second input is the value, which will be choosed if the statement is null
You use this function, if you want to search for null values in your tables.
