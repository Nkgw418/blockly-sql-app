Alias variable

The block

This block symbolises the alias of a subselect, subselect in the 'WHERE'-section or of the groupfunctions. An alias in SQL is a name for a result column. It is used as a value in the "ORDER BY" or "GROUP BY" -clause .

Functionality

At first you need to set an alias. Either you set the alias of a subselect or of one of the groupfunctions.
For more information about how to set an alias, click here. After setting the alias the variable will be filled automatically with the alias, which the alias. If you set more then one alias, the first one will be default value. If you want to change the alias, because you defined more than one. , you can klick on the arrow in the variable field and choose the aliasname as you see in the picture on the right.
