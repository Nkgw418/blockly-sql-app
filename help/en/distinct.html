Distinct

The block

This block symbolises the distinct clause. Sometimes a column can contain many duplicate values. If you want to list only the different values, you use distinct. It is used as a value in the "SELECT"-section and in some of the groupfunctions. The block changes also the colour to represent types.

Functionality

As you see in the picture on the right, this block is used with one table and column block. Notice that there can be only one table and column block for each distinct clause.

The types

This colour ,apple green, symbolises list, like all selected columns.

This colour,violet, symbolises the numeric values of the database

This colour, brown, symbolises the textual values of the database

This colour,tapestry, symbolises the date values of the database

This colour ,cyan, symbolises the boolean values of the database
