Text variable

The block

This block symbolises textual variables. It is used as a value in the compare operator, in the "IF" and "IF NULL" of the otherfunction, in charfunctions.

Functionality

Per default the variable is empty, but if you want to use it you need to fill it.

As you see in the picture on the right, this block has an included text field, where you can change the value.

If there is somthing wrong with your value, like it's empty, there will be an alert. This is shown in the picture to the left.
